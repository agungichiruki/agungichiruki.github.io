3 - Menjalankan Kode Python

03 Mei 2022

|

#agungichiruki

Untuk menjalankan kode Python, buka terminal dan arahkan path-nya ke direktori tempat kita menyimpan kode Python yang akan kita jalankan. Kita akan menjalankan kode Python hello_world.py yang sudah dibuat. Ketikkan perintah dibawah ini untuk menjalankan kode Python :

python hello_world.py

Output : 
Hello World!

Perintah diatas menjalankan Interpreter Python untuk menterjemahkan dan mengeksekusi kode-kode Python yang ada pada file hello_world.py.
